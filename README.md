
BAWS SLUSH GENERATOR
====================

## Startup

```bash
npm -g i slush
npm -g i slush-baws

mkdir myProject
cd myProject
slush baws
```


## Use

```bash
cd myProject

source .aliases
b --help
```


## Hints

* use *lowercase* for app names
* use PascalCase for resource names
